*Модель предсказывающая жанры фильмов*<br>
**Описание модели** <br>
Модель обучена на датасете TMDB-5000, датасет английский и семилетней давности.
По полученным названию и описанию, она предсказывает жанры фильма. <br>
**Рабочий пайплайн**<br>
***1*** Вся подготовка данных происходит в data_preparation.ipynb.<br>
***2*** Обучение и построение модели в model.ipynb. (в обоих файлах уже есть комментарии к коду)<br> 
***3*** Файл export_artifacts.py превращает лайтнинговый чекпоинт в приемлемый для HF формат.
(для этого модуль с самой моделью я залил в model_module.py - там лежит копия ячейки из ноутбука model)<br>
***4*** <br>
Остается только залить на HF модель с помощью push_model.py и app и requirements с помощью push_app_and_requirements.
